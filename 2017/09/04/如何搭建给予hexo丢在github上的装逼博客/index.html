遇到的主要问题

###1.更换主题
可以去github上去找开源的hexo主题，下载\hexo\blog\themes文件夹下面即可，然后blog文件夹下修改配置文件_config.yml中的theme: *

###2.github Page
新的github这里修改了，百度都是一些老方法，新建的时候必须要导入别的库的代码，我就随便导入的，然后把代码down下来，然后全部删除上传提交，清空远程仓库。
然后把blog目录下的public文件下下的所有文件复制到本地仓库，在提交推送，就ok了。

###3.部署到github
在git bash中执行 npm install hexo-deployer-git –save
然后修改_config.yml中的
deploy:
type: git
repository: *
branch: master
注意冒号后面有空格,没有空格不会报错，但是执行hexo d以后没反应。
